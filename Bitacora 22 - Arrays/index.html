ARRAYS Y CICLOS DE JAVASCRIPT

Stop

Ingresa números indefinidamente hasta que ingrese la palabra “Stop”. Se almacenarán todos los datos en un array y se mostrarán al final.

    Dividir Arrays

    Ingresa números indefinidamente hasta que ingrese el número 0 (cero). Se creará un array para los números pares, otro para los impares y un tercero para almacenar datos no numéricos. Al final el ciclo mostrará los resultados.

    Eliminar extremos

    Ingresa números indefinidamente hasta que se ingrese el número 0 (cero). Almacenará en un array solamente los números y se eliminaran del array el número más alto y el más bajo.

    Dividir Arrays

    Ingresa cualquier carácter hasta que ingrese un número 0 (cero). Se almacenará cada ingreso en una posición del array y al finalizar dividirá la cantidad de elementos en 3 arrays iguales.
